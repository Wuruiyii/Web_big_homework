# 过往来信 Vue 3.0 · 粉色回忆站

## 运行方式

```bash
npm install
npm run dev
```

如果安装慢，可以先切换镜像源：

```bash
npm config set registry https://registry.npmmirror.com
npm install
npm run dev
```

## 本版优化点

1. 全站统一为粉色回忆录风格，保留首页、文章、图片墙、百宝箱、留言和 AI 对话。
2. 已删除“微言”页面、导航入口、首页微言预览和相关跳转。
3. 首页“把普通日子收藏起来”模块已放宽布局，整体留白更舒展。
4. 图片墙“九宫格收藏”已增加间距，照片卡片不再显得拥挤。
5. `与我对话` 页面已删除接入 Coze 的说明组件，页面本身不再纵向滚动，只保留聊天框内部滚动。
6. package.json 已固定依赖版本，避免 latest 版本导致安装不稳定。

## 页面说明

- 首页 `/`：封面轮播、文章流、作者卡、快捷入口、普通日子收藏模块。
- 文章 `/sort`：分类、标签、搜索、时间线、文章详情、收藏文章。
- 图片墙 `/travel`：照片筛选、主图切换、全屏预览、旅拍路线、音乐和视频。
- 百宝箱 `/favorite`：收藏入口、关于我、友人帐、弹幕墙、随机诗笺。
- 留言 `/message`：投递来信、情绪筛选、珍藏留言。
- AI 对话 `/talk`：与另一个“我”进行本地互动式对话。
